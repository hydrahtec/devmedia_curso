Variaveis

Uma varialvel é um espaço reservado na memoria do computador para se armazer informações que serão usadas em algum momento dentro do codigo. As variaveis se devidem em tipos, tambem conhecidos como tipos primitivos.

Importante:

  1. Maiusculas e minusculas fazem diferença.
  2. Ao se declarar uma varialvel não é necessario especificar seu tipo, a propria linguagem consegue fazer a definição.
  3. Utilize nomes que tenham a ver com o conteudo que será armazenado.

Tipos de variaveis (Primitivas)

  • Number

    • Não distingue valoreres numericos inteiros ou reais (fracionados, positivos ou negativos, essa variavel ingloba tudo)

    • infinity
    • NaN - (Significa que o valor não é um número.)

  • String

    • Trata-se de uma cadeia de caracteres, não necessariamente precisa ser um texto. Exemplo: o CPF é caracterizado como String pois possui alem dos numeros, a presença de pontos e traço.

  • Bulean

    • Essa variavel resebe os valores 'verdadeiro' ou 'falso'

  • Null
  • Undefined
  • Object
    • arrey - (Arrey ou vetor ou ainda variavel composta, é uma variavel que contem varios elementos. Cada elemento é composto por seu valor e por uma chave de identificação. Toda chave de identificação sempre começam com 0 - let var = [5,9,1])
      • Para adicionar um novo elemento em um novo espaço dentro do arrey basta declarar: var['3']= 7 (var= [5,9,1,7] Obs.: 'adicionar uma terceira posição e preenche-la com o elemento 7')
      • Para adicionar um novo elemento dentro do mesmo arrey sem precisar se preocupar com a posiçãp dele basta declarar: var.push(4) (var=[5,9,1,7,4]) Obs.: Enpurre para dentro da variavel o elemento 4 e crie uma nova posição para isso.
      • Para descobrir a quantidade de elementos de um arrey basta declarar: var.length que nesse caso retornaria (5).
      • Para ordenar os elementos do arrey basta declarar: var.sort() var= [1,4,5,7,9]
  • Function - (Por padrão as funções possuem chamada + parametros + ação + retorno Funções são ações que são exucutadas assim que são chamadas ou em em decorrencia de um evento.)

Operadores

Os operadores possuem uma ordem de precedencia para serem utilizados, primeiro serão feitas as operações aritimeticas depois as operações relacionais depois os operadores lógicos

Operadores aritimeticos

Estes operadores são usados para fazer calculos.

  • [+] - (Operador de adição)
  • [-] - (Operador de subtração)
  • [*] - (Operador de mutiplicação)
  • [/] - (Operador de divisão real - contem virgula)
  • [%] - (Operador de divisão inteira - sem virgula)
  • [**] - (Operador de potenciação)

Ordem de Precedencia dos operadores aritimeticos

  1. [()]- (Primeiro serão feitos as operações que estiverem entre parenteses.)
  2. [**] - (Segundo as operações de potenciação.)
  3. [* / %] - (Terceiro as operações de mutiplicação, divisão real e divisão inteira na ordem que aparecerem na equação.)
  4. [+ -] - (Quarto serão feitas as operações de soma e subtração.)

Operadores de atribuição

  • [=] - (Operador de atribuição simplies - var a = 5+3; var b= a + 4...)
  • [n=n+4]Operador de auto-atribuição. A variavel recebe ela mesma com um incremento.
  • [+= / -= / *=...]Opreradores de atribuição simplificada. usado quando a variavel recebe ela mesma com um incremento
  • [var++ / var--] - (Esse tipo de operador simplifica o codigo sempre que precisamos acrestar ou retirar 1 ditigo da variavel.)

Operadores relacionais

Os operadores relacionais tem como retorno padrão valores bulenos: true or false. Numa mesma equação os operadores aritimeticos tem prioridade na ordem de precedencia

  • [<] - (Menor)
  • [>] - (Maior)
  • [<=] - (Menor ou igual)
  • [>=] - (Maior ou igual)
  • [==] - (Igual - os valores devem ser iguais mas os tipos podem ser diferentes.)
  • [!=] - (Diferente)
  • [===] - (Identico - o valor e o tipo devem ser os mesmos.)

Operadores lógicos

Sua ordem de precedencia é a seguinte: primeiro o NAO, depois o E e por fim o OU (! - && - ||)

  • [!] - (Operador NAO -negação)
  • [&&] - (Operador E -Conjunção (binario, os 2 devem ser verdadeiros para ser true))
  • [||] - (Operador OU -Disjunção (binario, apenas um lado precisa ser verdadeiro para ser true))

Operadore ternário

Esse operador possui tres partes.

  • [teste '?' (valor se true) ':' (valor se false)] Realiza o teste, se verdadeiro '?' é usado o primeiro valor, se falso ':' será usado o segundo valor.

DOM (Document Object Model)

É a versão web do javaScript. Poderosa ferramenta que esta dentro do navegadorar. Um conjunto de objetos para o tratamento de componentes visuais, que auxiliam na criação de applicações.

É possivel acessar qualquer elemento da arvore DOM atraves de 5 formas: por marca, por ID, por name, por class e por seletor.

Selecionando por MARCA

  • [.getElementsByTagName()[0,1,2,3]] - (Seleciona um elemento pelo nome da tag. Como possivelmente pode haver tags repetidas é possivel indicar a posição que ela se encontra insirendo [...] sendo que o primeiro elemento vai estar na posição [0])

Selecionando por ID

  • [.getElementByid()] - (Seleciona elementos atravez do seu id.)

Selecionando por NAME

  • [.getElementsByName()[...]] - (Seleciona elementos atravez do seu nome.)

Selecionando por CLASS

  • [.getElementsByClassName] - (Seleciona elementos atravez da sua class)

Selecionando por SELETORES (CSS)

  • [querySelector(ex.: div#id ou div.class)] - (Faz a seleção atraves de componentes CSS; Funcionalidade nova, pode não funcionar em todos os navegadores)

Dicas e macetes

Operador [+]

Ele pode agir como adição ou concatenação (usado para juntar textos com variaveis ou variaveis com variaveis qunado queremos mostralas varios dados na tela). Se o valor de todas as variaveis forem numericos (será realizada a operação de adição) mas se uma das variaveis for do tipo string (Será realizado a concatenação)

Tamplete strings

Ao inves de usar o operador [+] para concatenar a exibição de textos e variaveis, o que deixa o codigo extenso e poluido, agora é possivel usar um recurso diferente que deixa o codigo mais limpo e organizado. Chama-se tamplate strings (ou formatador de strings), ele consiste em escrever o conteudo (textos e variaveis) que queremos apresentar ao usuario, todo dentro de uma mesma estrutura. ex: window.alert(`Texto ${var} texto, ${var} texto texto!!`). Neste caso todo o conteudo é posto dentro crases e as variaveis são chamadas conforme a necessidade sem precisar fazer a separação uma da outra, basta colocar as variaveis dentro da estrutura ${}.
